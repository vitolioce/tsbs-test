# Immagini Blocchi

Questa cartella contiene le immagini per i blocchi delle forme Tetris.

## Come personalizzare

1. Crea immagini PNG di 65x65 pixel (o altra dimensione quadrata)
2. Salva i file con i nomi dei colori:
   - `block-cyan.png` → Forma I (cyan)
   - `block-orange.png` → Forma L (orange)
   - `block-purple.png` → Forma T (purple)
   - `block-yellow.png` → Forma O (yellow)
   - `block-red.png` → Forma Z (red)
   - `block-green.png` → Forma S (green)
   - `block-blue.png` → Forma J (blue)
   - `block-pink.png` → Forma U (pink)

3. Opzionalmente, crea anche:
   - `block-valid.png` → Per posizione valida durante drag (verde)
   - `block-invalid.png` → Per posizione non valida durante drag (rosso)

## Strumenti consigliati

- **Photoshop / GIMP**: Per creare immagini professionali
- **Figma**: Design online
- **Aseprite**: Per stile pixel art
- **Online**: https://www.photopea.com (editor gratuito)

## Tips

- Usa bordi e ombre per effetto 3D
- Aggiungi texture per realismo
- Mantieni trasparenza sui bordi per arrotondamenti
- Dimensione consigliata: 128x128px (verrà scalata)
